Cellular Automata: Visual and Interactive Simulations

Overview

This project explores elementary cellular automata using two complementary approaches: a batch rendering script that generates high-resolution rule evolutions as images, and a graphical user interface (GUI) that lets users input a rule and watch it evolve in real-time.

Implemented Models

  • cell10rules.py — Generates 50+ step rule evolutions (Rules 30, 110, etc.) and outputs them as PNG files using matplotlib.
  • cellular.py — A Tkinter GUI that allows user input for rule, width, and steps. The simulation animates row-by-row with scroll support and reset options.

Results and Exploration

By adjusting parameters like grid width and number of steps, the simulations reveal:

  • Chaotic patterns (e.g., Rule 30)
  • Complex, structured growth (e.g., Rule 110)
  • Emergence of order from randomness

Sample output files include:

  • Rule110.png
  • Rule110_wide.png
  • Rule110_steps500.png

Research Insight

Elementary cellular automata model how simple binary rules can produce surprisingly intricate behavior. These systems are fundamental to complexity science, showing how computation can emerge from simple units.
