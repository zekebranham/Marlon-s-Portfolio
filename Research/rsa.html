RSA Public Key Cryptography Implementation

Overview

This project is a Python-based RSA encryption and decryption tool created from scratch, designed to demonstrate how public and private key cryptosystems work. It showcases prime number generation, key creation, and message encryption with block-based encoding.

Features

  • Generates secure prime numbers using sympy
  • Implements keypair generation (public and private)
  • Encrypts multi-byte plaintext messages with block splitting
  • Decrypts ciphertext using modular arithmetic

Sample Flow

Input alphabetic message to encrypt with RSA: hello

p: 130945...
q: 137201...
n: 179688...
z: 179687...
e: 41273
d: 146557...

Ciphertext: 735281...
Plaintext: hello

Educational Goals

The goal of this project was to implement RSA from the ground up to learn how modular arithmetic, prime theory, and cryptographic systems interact. Unlike black-box libraries, this version shows how e, d, and n are generated and applied directly.
