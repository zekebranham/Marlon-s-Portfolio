Shell Security & Exploit Research

Overview

This section highlights hands-on experiments involving classic and modern shell vulnerabilities, including exploits like Shellshock, XSS, CSRF, and buffer overflows. These projects were completed within a virtualized Linux environment using Oracle VirtualBox.

  • Shellshock: environment variable injection
  • Cross-Site Request Forgery (CSRF)
  • Cross-Site Scripting (XSS) worm payloads
  • Buffer Overflow analysis and payload crafting
  • Encryption and hashing via shell scripting

Exploit Research Highlights

  • Shellshock Exploit: Injected Bash functions through HTTP headers to execute arbitrary shell commands.
  • CSRF Protection: Implemented token and same-site cookie validation strategies.
  • XSS Worm: Built and tested a JavaScript-based self-replicating worm for educational purposes.
  • Buffer Overflow: Manually crafted and debugged C payloads to overwrite memory and escalate control.
