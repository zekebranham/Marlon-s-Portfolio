CompTIA Security+ Certification Journey

Overview

This section documents my ongoing journey to earn the CompTIA Security+ certification. Through this process, I have gained practical and theoretical knowledge in cybersecurity principles, systems configuration, incident response, risk management, and cryptographic technologies.

Learning Resources

  • Professor Messer's Security+ Training Series
  • CompTIA Security+ Study Guide (SYO-601)
  • TryHackMe & practice labs (threat modeling, firewall configs)

Topics Studied

  • Types of Threat Actors and Vulnerabilities
  • Security Assessments and Penetration Testing
  • Secure Network Architecture and Systems Design
  • Cloud and Virtualization Security
  • Authentication, Authorization, and Identity Management
  • Cryptography and PKI
  • Incident Response & Forensics Basics

Labs and Exercises

As part of my preparation, I conducted hands-on labs using both Linux and Windows environments:

  • Simulated brute-force login detection
  • Firewall and port filtering configuration in Windows Server
  • Analyzed logs using SIEM-like tools (e.g., Splunk)
  • Configured secure file sharing via SMB and ACLs

Next Steps

I’m continuing toward full exam readiness by combining structured study with real-world practice. Once certified, I will use this as a foundation to pursue more advanced certifications (such as CySA+, CASP+, or CISSP).

Reflection

Preparing for Security+ has strengthened my confidence in defending networks and systems. It's helped me bridge the gap between conceptual knowledge and hands-on application — especially in areas like access control, secure configurations, and network monitoring.
