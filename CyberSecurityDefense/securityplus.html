CompTIA Security+ Certification Journey

Overview

In March 2025, I earned the CompTIA Security+ (SY0-701) certification. This section documents my journey preparing for and passing the exam. Along the way, I gained both theoretical and hands-on experience in cybersecurity principles, systems configuration, incident response, risk management, and cryptographic technologies.

Learning Resources

  • Professor Messer's Security+ Training Series
  • CompTIA Security+ Study Guide (SYO-601)
  • Practice labs (threat modeling, firewall configs)

Topics Studied

  • Types of Threat Actors and Vulnerabilities
  • Security Assessments and Penetration Testing
  • Secure Network Architecture and Systems Design
  • Cloud and Virtualization Security
  • Authentication, Authorization, and Identity Management
  • Cryptography and PKI
  • Incident Response & Forensics Basics

Labs and Exercises

As part of my preparation, I conducted hands-on labs using both Linux and Windows environments:

  • Simulated brute-force login detection
  • Firewall and port filtering configuration in Windows Server
  • Analyzed logs using SIEM-like tools (e.g., Splunk)
  • Configured secure file sharing via SMB and ACLs

Next Steps

After earning Security+, I plan to pursue additional hands-on experience in threat detection and system defense. I'm also exploring advanced certifications such as CySA+ and CISSP to further expand my skills in security operations and leadership.

Reflection

Preparing for Security+ has strengthened my confidence in defending networks and systems. It helped me bridge the gap between conceptual knowledge and hands-on application — especially in areas like access control, secure configurations, and network monitoring.
